# Changelog

Все важные изменения в проекте Fishertools будут документированы в этом файле.

## [0.2.0] - 2025-01-25

### 🚀 Крупное обновление - Новая концепция библиотеки

**Новая миссия:** "Инструменты, которые делают Python удобнее и безопаснее для новичков"

### ✨ Новые возможности

#### 🚨 Система объяснения ошибок
- **Главная функция:** `explain_error()` - объясняет ошибки Python на русском языке
- **Поддерживаемые ошибки:** TypeError, ValueError, AttributeError, IndexError, KeyError, ImportError, SyntaxError
- **Структурированный вывод:** объяснение, совет по исправлению, пример кода
- **Форматирование:** консольный вывод с цветами, JSON, plain text

#### 🛡️ Безопасные утилиты
- `safe_get()` - безопасное получение элементов из коллекций
- `safe_divide()` - деление без ошибки на ноль
- `safe_max()`, `safe_min()`, `safe_sum()` - безопасные математические операции
- `safe_read_file()`, `safe_write_file()` - безопасная работа с файлами
- Все функции предотвращают типичные ошибки новичков

#### 📚 Обучающие инструменты
- `show_best_practice()` - показывает лучшие практики Python
- `generate_example()` - генерирует примеры кода для изучения
- `list_available_concepts()` - список доступных концепций
- Образовательный контент на русском языке

### 🏗️ Архитектурные изменения

#### Новая модульная структура:
```
fishertools/
├── errors/          # Система объяснения ошибок
├── safe/           # Безопасные утилиты
├── learn/          # Обучающие инструменты
└── legacy/         # Функции для обратной совместимости
```

#### Чистый API:
- Основной импорт: `from fishertools import explain_error`
- Модульные импорты: `from fishertools.safe import safe_get`
- Все старые функции доступны через `fishertools.legacy`

### 🔄 Обратная совместимость
- **100% совместимость** - все функции из версии 0.1.0 сохранены
- Функции перенесены в модуль `legacy` с предупреждениями об устаревании
- Старые импорты продолжают работать: `from fishertools import utils, decorators, helpers`

### 🧪 Тестирование
- **163 теста** с полным покрытием функциональности
- **Property-based тестирование** с библиотекой hypothesis
- **12 свойств корректности** проверяются автоматически
- Интеграционные тесты для всех модулей

### 📖 Документация
- Полностью обновлённый README.md с примерами
- Документация на русском языке для всех функций
- Руководство по миграции для существующих пользователей
- Примеры использования для каждого модуля

### 🛠️ Технические улучшения
- Обновлён до Python 3.8+ (поддержка до Python 3.14)
- Современная структура проекта с pyproject.toml
- Улучшенная обработка ошибок с пользовательскими исключениями
- Оптимизированные зависимости

### 📦 Установка и использование

```bash
pip install fishertools==0.2.0
```

```python
# Новая основная функция
from fishertools import explain_error

try:
    result = 10 / 0
except Exception as e:
    explain_error(e)

# Безопасные утилиты
from fishertools.safe import safe_get, safe_divide

# Обучающие инструменты
from fishertools.learn import show_best_practice

# Старые функции (с предупреждениями)
from fishertools.legacy import hash_string, generate_password
```
